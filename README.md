# Motor Insurance Claims Dashboard 🛡️

An interactive dashboard analysing 500 motor insurance claims across Kenya (2022–2023), built with Python and Plotly.

## What It Shows
- Monthly claims volume over time
- Claim status breakdown (Settled, Pending, Rejected)
- Total claims by region across 8 Kenyan counties
- Average claim amount by vehicle type
- Monthly total claim amounts in KES
- Claimant age distribution

## Key Metrics
- **Total Claims:** 500
- **Average Claim:** KES 101,718
- **Rejection Rate:** 15%
- **Most Claims:** Nairobi region

## Tools Used
- Python
- Pandas (data manipulation)
- Plotly (interactive visualisation)

## How to Run
1. Clone this repository
2. Install dependencies: `pip install pandas plotly`
3. Run: `python dashboard.py`
4. Dashboard opens automatically in your browser
